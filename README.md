# Telegram Bookmarks

<p> Share any interesting links, texts, pictures, videos to a Telegram group with topics (e.g. Dev, Books, Read Later, Sports, Watch, Learning etc.). 

This app provides a frontend for this group with topics listed in the left, and each of the message listed as a card in the right.</p>

> [!]
> Code mostly generated by Antigravity + Claude Opus 4.5

- Telegram Groups-Topics as the backend
- Manual Sync
- 2-panel + cards layout

## Tech Stack:
- FastHTML
- HTMX
- SQL Lite
- [Telethon](https://github.com/LonamiWebs/Telethon) Python 3 library for Telegram API access
- [uv](https://github.com/astral-sh/uv)
- Docker

## Run:

Initial Telegram Login and sync including media (Warning: Sync is slow, not optimized, just a single threaded full sync)

1) > uv run python telegram_client.py

    Optional parameters

    --d - download media (else messages only)

    --p - download pictures only (no videos)

2) > uv run python main.py

    opens a local server at http://localhost:5001

3) Or alternatively run with docker:

    > docker build -t tgmarkers:1.0 . <br>
    > docker tag tgmarkers:1.0 tgmarkers:latest <br>
    > docker compose up -d
